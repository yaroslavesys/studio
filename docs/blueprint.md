# **App Name**: AccessFlow

## Core Features:

- Google OAuth2 Authentication: Secure user authentication using Google OAuth2 within the `@trafficdevils.net` domain.
- User Role Management: Define and manage user roles (User, TechLead, Admin) with distinct permissions.
- Department Management: Create and manage departments within the organization.
- Access Request Submission: Allow users to submit access requests with title, description, and request type (system, data, other).
- Access Request Approval Workflow: Enable TechLeads to approve or reject access requests from their department.
- Admin Control Panel: Provide admins with full control over users, departments, and access requests.
- Dashboard Summarization Tool: Summarize the information displayed on the dashboard with a generative AI tool.

## Style Guidelines:

- Primary color: Crimson (#DC143C) for a bold and energetic feel.
- Secondary color: Black (#000000) for contrast and a modern look.
- Accent color: Firebrick (#B22222), a darker shade of red, to highlight key elements.
- Body and headline font: 'Roboto', a sans-serif font, for readability and a clean aesthetic.
- Use sharp, geometric icons in white or light gray to stand out against the dark background.
- Employ a high-contrast layout with dark backgrounds and red accents to create a visually striking interface.
- Use bold and dynamic animations to enhance the user experience and create a sense of urgency.
